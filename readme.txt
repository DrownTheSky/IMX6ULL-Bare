【*】程序简介

-工程名称：Bare(IMX6ULL芯片裸机历程)
-实验平台: 100ask_imx6ull 开发板 
-实验代码：Makefile实现工程管理，startup_imx6ull.S实现IRQ中断的处理

【 ！】功能简介：

key1轮询翻转LED，key2中断翻转LED


【 ！】实验操作：

在linux环境下使用 arm-linux-gnueabihf- 编译器
make 编译工程
make clean 清除工程
make print 打印相关变量

.bin 源代码二进制文件
.dis 反汇编文件
.imx IMX6ULL芯片专属可执行文件，可下载到外部DDR中
.img 镜像文件可下载到SD卡或eMMC中
